Vim es un editor de texto altamente configurable y eficiente, utilizado directamente en la terminal. Aunque no es una herramienta de Bash en sí mismo, es el editor predeterminado en muchos sistemas Unix y Linux, lo que lo convierte en una herramienta fundamental para cualquier usuario de Bash que necesite editar archivos de texto, scripts o archivos de configuración.

Vim opera en diferentes modos que dictan qué hace cada pulsación de tecla. 

Los modos principales son:
- Modo normal : Es el modo predeterminado al abrir Vim. Se usa para navegar por el archivo, borrar, copiar, pegar y realizar otras acciones de edición sin insertar texto.
- Modo insertar : Para entrar en este modo se debe de insertar una tecla como i. Aquí, Vim se comporta como un editor de texto. Para salir y volver al modo normal, presiona ESC. 
- Modo Visual Se usa para seleccionar bloques de texto de manera similar a como lo harías con el mouse. Puedes seleccionar líneas, palabras o columnas.
