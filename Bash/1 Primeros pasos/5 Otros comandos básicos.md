
```
whoami
```
Muestra el usuario actual.

```
cal
```
Muestra el calendario.

```
date
```
Muestra la fecha y hora actuales.

```
uptime
```
Tiempo encendido del sistema.

```
hostname
```
Nombre del equipo.

```
uname 
```
Información del kernel.

```
uname -a
```
Información del kernel/sistema.

```
clear
```
Limpia la pantalla (no borra)
