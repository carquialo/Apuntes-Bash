Para cambiar de directorio utilizamos el comando ``cd``

``dir`` hace referencia al directorio

```
cd dir
```
Te desplaza al directorio inferior indicado.

```
cd dir/dir/dir
```
Te desplaza varios directorio a la vez.

```
cd ..
```
Te desplaza al directorio superior al actual.

```
cd ../../../../
```

Para desplazarte varios niveles superiores a la vez.

```
cd .
```
Hace referencia al directorio actual.

```
cd ~
```
Te desplaza directamente a tu directorio personal (home).

IMPORTANTE
Utiliza la tabulación como ayuda para completar automáticamente comandos o visualizar archivos o directorios sugeridos.
