`cat` : Muestra el contenido de un archivo (ideal para archivos pequeños).

`less` : Se utiliza para archivos más largos, ya que permite la paginación del contenido. 
Con less el usuario está dentro de un editor en modo visualización. Con "q" para salir.

`more` : Es similar a less pero con menos funcionalidad.

Para ver una porción de un archivo, se utiliza comando como : 

`head` : Muestra por defecto las primeras 10 líneas. Se puede poner como parámetro `-n "número de página"` 

`tail` : Muestra las últimas 10 líneas de un archivo. Lo contrario que head.
