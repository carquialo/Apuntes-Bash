El ``sistema de gestión de archivos Unix`` es una estructura jerárquica que organiza y gestiona los archivos y directorios.

``/`` : Es la raíz. Es el directorio donde cuelga los demás directorios.
``/home`` : Es donde se guarda las carpetas personales del usuario.
``/etc`` : Almacena los archivos de configuración del sistema.
``/bin`` :  Contiene los programas básicos del sistema.
``/usr`` : Usualmente guarda software adicionales y utilidades para los usuarios.
``/var`` : Contiene archivos variables, como registros del sistema (logs) y colas de mensajes. 
``/tmp`` : Almacena archivos temporales que el sistema puede borrar periódicamente.
