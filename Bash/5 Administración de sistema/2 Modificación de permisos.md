El comando principal para esta tarea es chmod. 

Hay dos métodos comunes para usarlos: 

1. Notación Simbólica: Este método usa letras. 
	- + : agrega permisos.
	- - : quita permisos.
	- = : Establecen los permisos como se especifican.
2. Notación Octal: Usa número para representar los permisos
	- rwx = 4+2+1 = 7
	- rw = 4+2+0 = 6
	- r-x = 4+0+1= 5
